bool StringUtils::isEOL ( char  character) [static]

Determines if character is (as considered in PDF specification) a end-of-line marker which means if it has one of these CR, LF. According to PDF specification the eol marker can be even combination CR+LF but in this case of boolean function that determines if it is EOL it is enogh to check just those two chars.

Returns:

char * StringUtils::strStrModified ( char *  strSource,
char *  searchString,
int  length = -1,

) [static]

This is byte modified strstr() method.
The only difference between strstr() is that in this method strSource can contain NULL bytes (NULL byte normaly ends the string) and searching does not end but searches further till the length is reached.
If length parameter is not specified (or has value < 0) than strlen of strSource is used.

Parameters:
strSourceSearch source
sarchStringString to be found in strSource
lengthLength of the strSource
caseInsensitiveSet to true if case insensitive search is needed.
Returns:
Pointer to the first occurence of the searchString in strSource. NULL if not found.
